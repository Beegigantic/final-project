▶History

1989–1990: The Conception of HTML

1989:
Tim Berners-Lee, a British scientist at CERN, proposed a system for sharing information using hypertext.
He called it the World Wide Web.
1990:
Berners-Lee developed the first version of HTML (HyperText Markup Language) alongside the first web browser
(WorldWideWeb) and the first web server (httpd).

1991: Public Introduction of HTML

August 1991:
Berners-Lee published the first documentation of HTML on the World Wide Web. It was simple and designed to
create and link documents.

1993: HTML+ Proposal

November 1993:
The HTML+ proposal aimed to extend HTML’s capabilities to include tables, forms, and other features.
However, it was never fully implemented.

1995: HTML 2.0

1995:
The Internet Engineering Task Force (IETF) published HTML 2.0, the first standardized version of HTML.
It included all features from earlier versions and added support for forms.
At this point, HTML began its rapid adoption as the foundation of web development.

1997: HTML 3.2 and HTML 4.0

January 1997:
The World Wide Web Consortium (W3C) released HTML 3.2, introducing:
Support for tables and applets
Cascading Style Sheets (CSS) for improved styling
December 1997:
HTML 4.0 was released. It included:
Support for dynamic content via scripting languages (e.g., JavaScript)
Frames and inline frames (iframes)

1999: HTML 4.01

December 1999:
HTML 4.01 was introduced as a minor revision, fixing issues in HTML 4.0.
This version remained the standard for over a decade as web technologies evolved around it.

2008: Introduction of HTML5

2004:
The development of HTML5 began as a response to the limitations of HTML 4.01. The Web Hypertext Application
Technology Working Group (WHATWG) initiated this effort.
2008:
A draft of HTML5 was released, marking a significant departure from previous versions. It aimed to:
Support multimedia (audio and video) without third-party plugins like Flash
Introduce APIs for offline storage and geolocation

2014: HTML5 Becomes the Standard

October 2014:
W3C officially recommended HTML5 as the new standard. It became the foundation of modern web development,
with features like:
Native multimedia support
Enhanced forms
Improved interactivity through APIs

2020: HTML Living Standard

2017:
The W3C and WHATWG merged efforts, making HTML a living standard maintained by WHATWG.
2020:
HTML continued evolving incrementally, with updates delivered in real time rather than waiting for major version releases.
HTML, alongside CSS and JavaScript, forms the backbone of web development. Its flexibility and ease of use have made
it integral to creating websites, web applications, and interactive content. As of today, HTML continues to evolve,
adapting to new technologies and user needs while remaining backward-compatible.
