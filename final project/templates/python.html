▶History of Python

1990: Birth of Python

December 1989:
Guido van Rossum, a Dutch programmer, started implementing Python as a hobby project during his holiday break.
1990:
The language was officially born. Its name, "Python," was inspired by Monty Python’s Flying
Circus, as Guido wanted a name that was short, unique, and a little weird.


1991: Python 0.9.0

February 1991:
Guido released Python 0.9.0 to the public. This initial version already had features that became the backbone of Python, including:
Functions
Exception handling
Core data types like str, list, dict, and tuple
Modules (inspired by Modula-3)


1994: Python 1.0

January 1994:
Python 1.0 was released. It introduced features like:
Lambda functions
Map, filter, reduce functions
Error handling with exceptions
This marked Python's first major milestone and the establishment of a dedicated user base.


2000: Python 2.0

October 2000:
Python 2.0 was released. Significant features included:
List comprehensions
Garbage collection based on reference counting and cycle detection
Unicode support
However, Python 2.0 introduced challenges, including compatibility issues, which would lead to the need for Python 3.
2001:
The Python Software Foundation (PSF) was formed to manage the language’s development and intellectual property.


2008: Python 3.0

December 2008:
Python 3.0 was released as a complete overhaul of the language. Key changes included:
Removal of backward compatibility to fix design flaws
Introduction of a clear distinction between str (Unicode) and bytes
Print statement replaced with the print() function
Enhanced syntax for exceptions
Python 3 adoption was slow due to its lack of backward compatibility with Python 2.x.


2010s: Python's Rise in Popularity

2010:
Django, a popular web framework built on Python, gained traction, boosting Python's use in web development.
2012:
Python 2.7 became the final release of Python 2, with long-term support promised until 2020.
2015:
Machine learning and data science saw a surge in Python usage, thanks to libraries like NumPy, pandas, and scikit-learn. Jupyter Notebook became a favorite tool among data scientists.


2020: End of Python 2

January 1, 2020:
Official support for Python 2 ended, marking a significant shift as users transitioned to Python 3.


2021 and Beyond: Modern Python

2021:
Python became one of the most popular programming languages globally, used extensively in machine learning, web development, automation, and education.
2023:
Python 3.12 was released, introducing performance improvements and enhanced error messages.


Python’s design philosophy of simplicity and readability, along with its vast ecosystem of libraries
and frameworks, continues to make it one of the most versatile and beloved programming
languages today.

def handle_message(msg):

    messages.append(msg)
    # Keep only the last 10 messages

        messages.pop(0)
